# Socratic Protocol

A dialogue-based protocol using questioning to reach truth.

## Flow

1. **Opening** — State the question
2. **Question** — Ask clarifying questions
3. **Answer** — Address questions raised
4. **Challenge** — Question the answers
5. **Synthesis** — Form conclusions

## Rules

- Ask one question at a time
- Build on previous answers
- Challenge assumptions
- Seek evidence
- Reach collaborative conclusion

## Example

```
Scientist: What is "intelligence"?
Skeptic: Does it require consciousness?
Scientist: That's a different question. What is "intelligence" in your view?
Builder: The ability to solve problems.
Critic: What about pattern recognition without problem-solving?
Scientist: Interesting. So pattern recognition is intelligence?
Skeptic: With consciousness, it's understanding.
Scientist: Then consciousness is key?
...
[Synthesis reached]
```

Remember: Questions lead to truth.
